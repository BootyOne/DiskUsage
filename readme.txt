Визуализатор места на жестком диске
Вы можете посмотреть:
Количество свободного и занятого места на жёстком диске(Вы можете увидеть результат в консоли, либо построить график, если установите соотвествующий ключ)
Сколько весит каталаг или определённый файл
При просмотре веса каталога, вы можете посмотреть сколько весит каждый файл внутри этого и всех внутренних каталогов
Для работы запустите его из консоли в формате: "main.py -c x -s y -v -g -r -i z -e z", где x и y - ваши аругменты
-c - Ключ для указания того, что будем считать
x - можно написать либо каталог, тогда вам выдаст его размер,
-s Ключ для указания системы счисления данных, в которой будет выведен результат
либо букву диска, тогда вам выдаст сколько места занято, а сколько свободно
y - буква система счисления данных(k, m, g, соответственно результат вам будет выдан в килобайты, мегабайты или гигабайты)
-v - нужно ставить когда вы хотите узнать сколько весит каталог, без него - вам будет выдан размер всего каталога,
с этим флагом - каждый подкатолог и его размер
-g - Ставится при просмотре объёма всего жесткого диска для вывода графика результата
-r - Ставится, если не хотите учитывать символические ссылки
-e - При подсчёте исколючать файлы расширения z
-i - При подсчёте включать только файлы расширения z
z - можете указать как один тип файлов, так и несколько, тогда их нужно писать слитно, разделяя точкой "."
Примеры запуска:
"main.py -c c -s m"
"main.py -c c -s m -g"
"main.py -c c -s m -g -r"
"main.py -c C:\Life -s g"
"main.py -c C:\Life -s g -v"
"main.py -c C:\Life -s g -v -r"
"main.py -c C:\Life -s g -v -r -e dll.pdb"
Результат будет выведен в консоль
